## 图片格式要求
- 使用PNG格式（避免JPEG压缩失真）
- 建议分辨率不超过800x600像素
- 文件大小控制在1MB以内
- 使用描述性的文件名

## 如何生成截图
1. 在你的程序中按'S'键保存截图
2. 或者使用屏幕截图工具
3. 将图片放在这个images/目录下
4. 确保文件名与HTML中引用的名称一致
